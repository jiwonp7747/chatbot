Phase 1 — Project Foundation

  • Vite + Vue 3 + TypeScript 프로젝트 초기화
  • Pinia 상태 관리 도입 — useState에서 defineStore로
  • 환경변수 체계 전환: REACT_APP_* → VITE_*

Phase 2 — Component Architecture

  • <script setup> 단일 파일 컴포넌트로 재작성
  • React hooks → Vue Composables 패턴 전환
  • 마크다운 렌더링: markdown-it + highlight.js 통합
